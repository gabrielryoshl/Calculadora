# 🧮 Calculadora 2000

Projeto simples de uma calculadora em Java que realiza as 4 operações básicas:

- ➕ Soma
- ➖ Subtração
- ✖️ Multiplicação
- ➗ Divisão

## 💻 Sobre o projeto

A aplicação funciona via terminal e permite ao usuário escolher a operação desejada, inserir dois números e visualizar o resultado.

Também possui:
- Validação de divisão por zero
- Menu interativo
- Opção de continuar ou encerrar o programa
